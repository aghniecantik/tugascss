Belajar CSS

Apa Itu CSS?

Cascading style sheet (CSS) merupakan sekumpulan aturan yang menyatakan bagaimana style diaplikasikan ke tag-tag HTML di dalam dokumen. Selain mengulas tentang pemanfaatan CSS, bagian ini juga menekankan pada pembuatan desain dengan menggunakan division. Bagaimanapun, untuk menghasilkan desain halaman web yang baik, penggunaan tabel tidak disarankan dan sebagai gantinya adalah dengan memanfaatkan division.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline Styles : Metode ini digunakan dengan cara memasukkan file css ke dalam tag HTML secara langsung.
  2. Internal/Embedded Styles : Menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Styles : Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dukumen HTML
